# dynam.space
A dynamics simulator and data collector. 

[https//dynam.space](https://dynam.space/) 

## Instructions
Load static website:
```
cd web
livereload
```

Edit demos:
```
cd web/d
vim torus.html
```

Javascript source code:
```
https://dynam.space/js/dynam.space.js
```
